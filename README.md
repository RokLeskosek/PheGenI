# PheGenI

Genetika izobrazbe
==================

Analiziral bom podatke povezane z geni povezanimi z izobrazbo. Zajel bom podatke s strani NCBI (https://www.ncbi.nlm.nih.gov/gap/phegeni).
Za vsak gen bom zajel:
* Okrajšavo imena
* Ime
* Lokacija
* Polimorfizem, ki je bil povezan z izobrazbo
* p-vrednost (stopnja zna?ilnosti)
* kje je gen najbolj izražen
* povezane bolezni
* povezani fenotipi

Delovne hipoteze:
* Je izobrazba genetsko povezana z drugimi fenotipi
* Obstaja povezava med lokacijo v genomu in omenjenimi geni?
* Je izobrazba genetsko povezana s katerimi boleznimi?
