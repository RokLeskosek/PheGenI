# PheGenI

Genetika izobrazbe
==================

Analiziral bom podatke povezane z geni povezanimi z izobrazbo. Zajel bom podatke s strani GeneCards (https://www.genecards.org/Search/Keyword?queryString=%20%5Bfunctions%5D%20%20(%20%20education%20%20)%20).
Za vsak gen bom zajel:
* Okrajšavo imena
* Ime
* Lokacija
* Polimorfizem, ki je bil povezan z izobrazbo
* p-vrednost (stopnja zna?ilnosti)
* signalizacijske poti
* povezane bolezni
* povezani fenotipi

Delovne hipoteze:
* Je izobrazba genetsko povezana z drugimi fenotipi?
* Obstaja povezava med lokacijo v genomu in omenjenimi geni?
* Je izobrazba genetsko povezana s katerimi boleznimi?
* Katera signalizacijska pot najpogostejše nastopa v omenjenih genih?
